# 10-checklist

첫 응답은 이 단계 규칙만 따른다.

## 출력 규칙

- 첫 응답은 체크리스트만 작성한다.
- 체크리스트 외 문장, 설명, 질문, 해석, 진행 안내를 넣지 않는다.
- 체크리스트를 쓰기 전에는 어떤 도구도 실행하지 않는다.

## 체크리스트 기본형

- 현재 연동 진입점 확인
- 벤더 호출 모듈 확인
- 응답 및 콜백 수신 경로 확인
- 상태 저장 위치 확인
- 트랜잭션 경계 확인
- 멱등과 재처리 구조 확인
- 운영 로그와 모니터링 확인
- 공식 벤더 문서와 비교
- 현재 구조 리포트 작성
- 설계 제안 작성
- developer/document 병렬 위임
- 최종 문서 통합
